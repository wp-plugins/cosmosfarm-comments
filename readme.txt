== Description ==

[코스모스팜 소셜댓글 홈페이지](http://www.cosmosfarm.com/plugin/comments)

**아주 쉬운 사용방법**

전문가가 아니어도 괜찮습니다. 단 몇 줄의 코드를 추가하는 것만으로도 당신의 웹 사이트에 댓글 플러그인을 설치할 수 있습니다. 즉시 더 많은 사용자들의 이야기를 들을 수 있습니다.

**모두가 참여 가능한 커뮤니티**

내 사이트의 회원과 소셜회원 모두 참여가 가능한 커뮤니티를 구축할 수 있습니다. 네이버, 카카오톡, 페이스북, 트위터, 구글 등 다양한 소셜로그인을 지원하며 내 사이트 회원까지 토론에 참여할 수 있어 더 많은 사용자를 확보할 수 있습니다.

**강력한 리얼타임 채팅**

실시간으로 댓글이 달려 페이지에 함께 있는 사용자끼리 채팅이 가능합니다. 화면을 보고 있지 않아도 단 하나의 대화도 놓치지 않도록 즉시 이메일로 알려드립니다. 사용자간 커뮤니티를 구축하기 위해서는 반드시 필요한 기능 입니다.

**편리한 부가 기능**

유튜브 주소만 입력하면 동영상 플레이어가 자동 생성되어 모바일 환경에서도 쉽게 동영상 공유가 가능합니다. 커뮤니티에 필요한 각종 편의 기능들이 지속적으로 추가됩니다.

**꼭 필요한 관리자 기능**

통합 댓글관리, 엑셀파일 다운로드, 관리자 알림, 비밀글, 로그인 API, 링크주소 차단, 스팸관리, 디자인 변경등 다양한 기능들을 손쉽게 관리할 수 있습니다.
